# smartisanBlog
🔨博客，基于锤子便签的博客程序。

## 说明
去年大概这个时候写的一个基于锤子便签的博客程序。今天翻出来还可以用，几个 API 保持和锤子便签一致，只需要抓取到相应路径下即可实现更新。

## 端部分使用说明


```sh
npm i // 安装必要的 npm 包
grunt // 生成前端线上版本
```

`www/config.js` 配置 API 地址
